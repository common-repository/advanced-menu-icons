Add custom SVG or image icons to WordPress menus. Customize icon size, position, color, and margins. Loads only used icons, optimizing site speed and design.

== Description ==

Advanced Menu Icons allows you to easily enhance your WordPress menus with custom SVG and image icons. Add icons from popular libraries or upload your own images for a unique and fully customizable menu design. This plugin provides advanced options for customizing icon positions, sizes, margins, and colors to perfectly match your website's style. Only the icons you use are loaded, optimizing site performance and keeping your menus sleek and professional.

== Features ==

- **SVG Icon Integration**: Load only the icons you need, optimizing performance with scalable and lightweight SVGs.
- **Multiple Icon Libraries**: Choose from an extensive range of icon sets, providing endless design possibilities.
- **Custom Image Icons**: Add your own custom images as icons, allowing full creative control over your siteâ€™s menu.
- **Flexible Icon Positioning**: Set your icons on the left or right of the menu items, adjusting for the perfect layout.
- **Icon Size Customization**: Easily adjust the size of your icons to fit any design need.
- **Margin Customization**: Set top, right, bottom, and left margins to give your icons the perfect spacing in any menu.
- **Custom Icon Colors**: Customize both the default and hover colors of your SVG icons to match your site's design.
- **No Unnecessary Load**: Only the icons you use are loaded, ensuring faster page loads and efficient site performance.

Take control of your WordPress menu design and make it stand out with advanced icon customization options.

== Installation ==

1. Download the advanced-menu-icons.zip file to your computer.
2. Unzip the file.
3. Upload the `advanced-menu-icons` directory to your `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Go to **Dashboard > Settings > Advanced Menu Icons** to start customizing your icons.

== Frequently Asked Questions ==

2. Unzip the file.

== Usage of Third-Party Services ==

### Reliance on External Services

This plugin relies on the use of a third-party service for fetching icons.

- **Service Name**: ReCorp API
- **Service Description**: This service is used to retrieve icon resources necessary for the plugin's functionality.
- **Service Link**: [ReCorp API](https://api.myrecorp.com/icons/)
- **Service Terms of Use**: [Terms of Use](https://api.myrecorp.com/icons/terms-of-use.html)
- **Service Privacy Policy**: [Privacy Policy](https://api.myrecorp.com/icons/privacy-policy.html)

### Data Transmission

When you use this plugin, data is sent to and received from the ReCorp API. This includes:

- **Data Sent**: File ID for retrieving the corresponding icons.
- **Data Received**: JSON data containing the icon resources.

By using this plugin, you agree to the terms and conditions outlined in the ReCorp API's terms of use and privacy policy.
